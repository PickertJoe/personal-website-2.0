Core Competencies

  • Frameworks/Runtimes: Rails, Django, Node.js
  • API Standards: REST, Graphql
  • Javascript: React, jQuery, StimulusJS
  • Devops: Terraform, AWS
  • General Tools: Git, Docker, VSCode
